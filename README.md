<h1>This is a website that sells clothes</h1>

<h3>Technology Used :</h3>
<h3>FE: ReactJs,Sass</h3>
<h3>BE: NodeJs/Express, MongoDb</h3>

<h2>Functions of the website</h2>
<p>I wrote CRUD APIs for clothing and users, implemented user authentication using JWT, displayed products, filtered products by color, size and category, allowed users to select the quantity of products, added products to the shopping cart, increase or decrease the number of products in the shopping cart and calculated the total price.
</p>
